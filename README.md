### Installation

- Just, Clone this repository -
  ```
  git clone https://github.com/CanPerest019/CanDork
  ```

- Now go to cloned directory and run `CanDork.py` -
  ```
  $ cd CanDork
  $ cd CanDork
  $ python3 CanDork.py
  ```

- On first launch, It'll install the dependencies and that's it. ***CanDork*** is installed.
